 # Clínica Médica 

O sistema modela o banco de dados de uma clínica médica, contemplando o cadastro de pacientes, médicos, consultas, prontuários e convênios. O objetivo e gerenciar de forma eficiente o fluxo de atendimento, histórico clínico dos pacientes e os planos de saúde aceitos pela clínica. 

O tema foi escolhido por permitir uma modelagem rica, com relacionamentos variados e uso de diferentes tipos de dados, refletindo um cenário real e relevante para sistemas de informação em saúde.
